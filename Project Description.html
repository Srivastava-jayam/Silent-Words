We intend to develop a smart wearable smart device for better communication between speech impaired and non-speech impaired community

Our System will convert sign language to Regional language(Speech) This will be a system which will also fulfill educational needs of that community by enhancing communication between both. In turn it will also help in creating a better bond of understanding between communities and will also educate others about sign language.

We are paying special attention about the product's having:


1.Focus on regional language conversion
2.Support for continuous and discontinuous gestures and speech
3.Portability and wearability
4.No Language barrier, a user-friendly interactive environment, especially in discussions, conversations and presentations.

Our Methodology

Hardware Development:

using Single Board Computer Cortex A8/Zynq, motor driver, DAC, Amplifiers and DAQ for signal processing, recognition and handling

Software Development:

adaptation (and or development) of language libraries, design and development of gesture recognition algorithm and database, developing Real Time OS (RTOS) for fast and easy recognition and GUI for end user.

Integration of the Co-design System:

After testing Hardware and Software performance separately, the integrating design methods will be adapted in this module. Then the performance analysis with the help of profiling, benchmarking and validation will be performed which will result in efficient system ready to be tested in the field.

Likely outcomes of the proposed project

A wearable device to convert Sign language to regional Speech.
Develop the local language APIs and digital directories
Develop the Gesture database and its meanings
Availability of the project deliverables and technical report online. Report and deliverables will be generated twice in a year which will provide the current status of the project.
The simulation model of the proposed project will be available on the same portal and which be shared on request to desired authority.
Collected data and algorithms would be available online on request.
