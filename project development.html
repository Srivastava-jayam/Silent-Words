We, a team of three second year undergraduate students under the mentorship of Dr. Abhishek Sharma have built and tested a machine learning algorithm which we name as “Sign Language to Regional language convertor”. We thought that a thing such as this should be there among us so that we can express our views as well as can understand the views of our fellow specially abled ones in a more effective manner.So for this purpose we developed a machine learnig algorithm which is running at the backend of this website ,it predicts the actions performed by te user in real time through the webcam and gives the output as words he/she is trying to say. We sincerely hope that we were able to serve that purpose well.
